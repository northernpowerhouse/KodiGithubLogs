Kodi Log Uploader Repository

This is the official repository for the Kodi Log Uploader addon.

Installation

  1. Download the repository zip: repository.kodiloguploader.zip
  2. In Kodi, go to Add-ons → Install from zip file
  3. Select the downloaded zip file
  4. Wait for the notification that the repository was installed
  5. Go to Add-ons → Install from repository → Kodi Log Uploader Repository
  6. Select Program add-ons → Kodi Log Uploader
  7. Click Install

Direct Installation

Alternatively, you can install the addon directly:

  1. Download: script.kodiloguploader-0.1.0.zip
  2. In Kodi, go to Add-ons → Install from zip file
  3. Select the downloaded zip file
